# Instagram MVP - 실행 방법

## 패키지 설치 완료!

Next.js 14와 모든 필요한 패키지가 설치되었습니다. ✅

## 개발 서버 실행 방법

### 방법 1: VSCode 터미널에서 실행 (권장)

1. VSCode에서 터미널 열기 (`Ctrl + ``)
2. 다음 명령어 실행:

```bash
npx next dev
```

### 방법 2: Windows PowerShell 또는 CMD에서 실행

1. PowerShell 또는 명령 프롬프트 열기
2. 프로젝트 폴더로 이동:
```bash
cd c:\Users\USER\Documents\GitHub\Instagram_mvp
```
3. 개발 서버 실행:
```bash
npx next dev
```

### 방법 3: 시스템 환경변수 PATH에 Node.js 추가 후 실행

현재 Node.js가 PATH에 등록되지 않아 일부 명령어가 작동하지 않습니다.

**PATH 추가 방법:**
1. 시스템 환경변수 설정 열기
2. Path 변수에 추가: `C:\Program Files\nodejs`
3. 컴퓨터 재시작
4. 이후 `npm run dev` 명령어 사용 가능

## 서버 실행 후

브라우저에서 접속:
- **주소:** http://localhost:3000

## 다음 단계

개발 서버가 실행되면 다음과 같이 개발을 시작할 수 있습니다:

1. 새로운 페이지 추가: `src/app/[페이지이름]/page.tsx`
2. API 라우트 추가: `src/app/api/[라우트이름]/route.ts`
3. 컴포넌트 추가: `src/components/`
4. TypeORM 설정: 데이터베이스 연결 및 엔티티 설정

## 문제 해결

서버가 시작되지 않으면:
- VSCode를 재시작
- 터미널을 닫고 다시 열기
- `npx next dev` 명령어 사용
